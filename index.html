INFO - Pada era perkembangan internet dari tahun 1990-an, teknologi web berkembang begitu pesat. Awalnya, web server dikembangkan hanya mampu melakukan komunikasi berbasis plaintext, sob. Tapi saat ini web server sudah mampu menampilkan konten-konten berbasis multimedia serta kemampuan menangani permintaan akses yang tinggi.

Web server merupakan layanan yang diberikan oleh komputer server berupa layanan penampil pemrosesan dari sebuah file berekstensi khusus website. Web server berjalan pada jalur TCP/IP pada port 80 dengan protokol bernama Hyper Text Transfer Protocol (http://) dengan aplikasi berbasis arsitektur client-server.

Pengertian Web Server

Web Server adalah sebuah aplikasi perangkat lunak yang bekerja pada sisi server untuk memberikan layanan website dan berjalan pada standar protokol TCP/IP, yaitu http:// pada port 80. Web server yang saat ini populer digunakan adalah apache yang mampu mengeksekusi file-file dengan ekstensi *.html, *.php, *.xml, *.css, *.js dan masih banyak lagi.

Cara Kerja Web Server

Setiap website yang tampil pada komputer personal memiliki proses yang cukup sederhana. Secara umum, cara kerja web server adalah menerima permintaan dari komputer klien melalui protokol http:// pada port 80 kemudian server akan memanggil layanan web dan mengecek path atau lokasi penyimpanan file website tersebut. Bila ditemukan, server akan mengirimkan file tersebut dengan memprosesnya terlebih dahulu untuk mengubahnya menjadi tampilan yang kemudian diterima oleh komputer klien. Pada peranan yang lebih luas, ada proses-proses yang melibatkan layanan publik seperti NS (Name Server) yang akan memberikan IP publik untuk akses website pada jaringan publik. Pada jaringan lokal, kita dapat menggunakan fitur virtual private server yang terdapat pada perangkat router yang kita miliki.
